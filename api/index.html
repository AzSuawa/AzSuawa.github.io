小小素API文档
上次更新: 2025-10-24

公开本API提供的数据应标注来源：azsu.top，普通请求存在频率限制，可向开发者申请访问密钥

BaseURL: https://api.azsu.top

MCPing

GET/mcping/json

返回指定MC服务器的Ping信息(JSON)

请求参数

    ip - 服务器地址

请求示例

https://api.azsu.top/mcping/json?ip=azsu.top

GET/mcping/image

返回指定MC服务器的Ping信息(图片)

请求参数

    ip - 服务器地址

    watermark - 自定义水印文字(可选)

请求示例

https://api.azsu.top/mcping/image?ip=azsu.top&watermark=§c延迟仅供参考§7｜§e小小素交流群:§6199755660§7｜§b制作:阿素本素§3(Az_Su)

Skin

GET/skin/file

返回指定ID由玩家上传的皮肤文件，若没有上传皮肤，正版ID返回正版皮肤，非正版ID返回菜鸟皮肤，用于azsu.top服务器设置皮肤功能。若SkinsRestorer插件无法识别请改为5003端口。

正版玩家皮肤来自Mojang API

请求参数

    player - 玩家ID

请求示例

https://api.azsu.top:5003/skin/file?player=Az_Su

状态码

HTTP状态码Code描述
20014000文件上传成功
40014101没有上传文件
40014102没有选择文件
40014103不支持的文件格式
40014104文件太大
40014105文件安全验证失败
50014201文件保存失败
50014202服务端缺失必要文件

2B2T.GG

GET/gg/tablist/json

返回2b2t.gg当前在线玩家列表(JSON)

该数据自2025.5.17起不再更新
https://api.azsu.top/gg/tablist/json

GET/gg/tablist/image

返回2b2t.gg当前在线玩家列表(图片)

该数据自2025.5.17起不再更新
https://api.azsu.top/gg/tablist/image

GET/gg/stats/<player_name>

返回指定玩家在2b2t.gg的信息

该数据自2025.5.17起不再更新

请求参数

    player_name - 玩家ID

https://api.azsu.top/gg/stats/Az_Su

GET/gg/rankings/player/<player_name>

返回指定玩家在2b2t.gg的排名信息

该数据自2025.5.17起不再更新

请求参数

    player_name - 玩家ID

https://api.azsu.top/gg/rankings/player/Az_Su

GET/gg/rankings/json/<type>/<player_name>

返回指定玩家在2b2t.gg中指定类型的排名信息

该数据自2025.5.17起不再更新

请求参数

    type - 类型：joins/kills/deaths/kd/playtime

    player_name - 玩家ID

https://api.azsu.top/gg/rankings/json/playtime/Az_Su

GET/gg/rankings/json/<type>

返回2b2t.gg中指定类型的排行榜前1000名

该数据自2025.5.17起不再更新

请求参数

    type - 类型：joins/kills/deaths/kd/playtime

https://api.azsu.top/gg/rankings/json/playtime

GET/gg/rankings/image/<type>

返回2b2t.gg中指定类型的排行榜前20名(图片)

该数据自2025.5.17起不再更新

请求参数

    type - 类型：joins/kills/deaths/kd/playtime

https://api.azsu.top/gg/rankings/image/playtime
