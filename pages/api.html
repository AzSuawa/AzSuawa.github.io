小小素API文档 2025.9.3

BaseURL: https://api.azsu.top

请求方式: GET

示例: https://api.azsu.top/mcping/image?ip=azsu.top

注意: 公开本API提供的数据需标注来源：小小素API或azsu.top

服务器状态查询

GET /mcping/json?ip=<ip>

返回指定MC服务器的Ping信息(JSON格式)

数据来源: Mojang服务器状态API

参数说明

ㅤip - 服务器地址(必需)

ㅤip - 服务器地址(必需)

示例URL

https://api.azsu.top/mcping/json?ip=azsu.top
GET /mcping/image?ip=<ip>&watermark=<水印>

返回指定MC服务器的Ping信息(图片格式)

数据来源: Mojang服务器状态API

参数说明

  • ip - 服务器地址(必需)
  • watermark - 自定义水印文字(可选)

示例URL

https://api.azsu.top/mcping/image?ip=azsu.top&watermark=小小素API

皮肤服务

GET /skin/skin/<玩家>

返回指定玩家上传的皮肤图片，用于阿素服设置皮肤功能

数据来源: Mojang皮肤API

参数说明

  • 玩家 - 玩家ID(必需)

示例URL

https://api.azsu.top/skin/skin/Steve

2B2T.ORG数据

GET /org/rank/json/playtime/<玩家>

返回指定玩家在2b2t.org近一个月游玩时长的排名信息

数据来源: 2b2t.vc API二次处理

参数说明

  • 玩家 - 玩家ID(必需)

示例URL

https://api.azsu.top/org/rank/json/playtime/Player1
GET /org/rank/json/kills/<玩家>

返回指定玩家在2b2t.org近一个月击杀数的排名信息

数据来源: 2b2t.vc API二次处理

参数说明

  • 玩家 - 玩家ID(必需)

示例URL

https://api.azsu.top/org/rank/json/kills/Player1
GET /org/rank/json/deaths/<玩家>

返回指定玩家在2b2t.org近一个月死亡数的排名信息

数据来源: 2b2t.vc API二次处理

参数说明

  • 玩家 - 玩家ID(必需)

示例URL

https://api.azsu.top/org/rank/json/deaths/Player1
GET /org/rank/json/all/<玩家>

返回指定玩家在2b2t.org近一个月的全部排名信息

数据来源: 2b2t.vc API二次处理

参数说明

  • 玩家 - 玩家ID(必需)

示例URL

https://api.azsu.top/org/rank/json/all/Player1

2B2T.GG数据

GET /gg/tablist/json

返回2b2t.gg当前在线玩家列表(JSON格式)

不再更新

参数说明

  • 无参数

示例URL

https://api.azsu.top/gg/tablist/json
GET /gg/tablist/image

返回2b2t.gg当前在线玩家列表(图片)

参数说明

  • 无参数

示例URL

https://api.azsu.top/gg/tablist/image
GET /gg/stat/<玩家>

返回指定玩家在2b2t.gg的信息

参数说明

  • 玩家 - 玩家ID(必需)

示例URL

https://api.azsu.top/gg/stat/Player1
GET /gg/rank/player/<玩家>

返回指定玩家在2b2t.gg的排名信息

参数说明

  • 玩家 - 玩家ID(必需)

示例URL

https://api.azsu.top/gg/rank/player/Player1
GET /gg/rank/json/<类型:joins,kills,deaths,kd,playtime>/<玩家>

返回指定玩家在2b2t.gg中指定类型的排名信息

参数说明

  • 类型 - joins/kills/deaths/kd/playtime(必需)
  • 玩家 - 玩家ID(必需)

示例URL

https://api.azsu.top/gg/rank/json/playtime/Player1
GET /gg/rank/json/<类型:joins,kills,deaths,kd,playtime>

返回2b2t.gg中指定类型的排行榜前1000名

参数说明

  • 类型 - joins/kills/deaths/kd/playtime(必需)

示例URL

https://api.azsu.top/gg/rank/json/playtime
GET /gg/rank/image/<类型:joins,kills,deaths,kd,playtime>

返回2b2t.gg中指定类型的排行榜前20名(图片)

参数说明

  • 类型 - joins/kills/deaths/kd/playtime(必需)

示例URL

https://api.azsu.top/gg/rank/image/playtime
