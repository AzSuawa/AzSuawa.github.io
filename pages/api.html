GET/mcping/image

返回指定MC服务器的Ping信息(图片)

请求参数

    ‎ ip - 服务器地址
    ‎ watermark - 自定义水印文字(可选)

Skin

GET/skin/file

返回指定ID由玩家上传的皮肤文件，若没有上传皮肤，正版ID返回正版皮肤，非正版ID返回菜鸟皮肤，用于azsu.top服务器设置皮肤功能。若SkinsRestorer插件无法访问请改为3326端口。

正版玩家皮肤来自Mojang API

请求参数

    ‎ player - 玩家ID

请求示例

https://api.azsu.top/skin/file?player=Az_Su

状态码

HTTP状态码 code字段 描述
200 14000 文件上传成功
400 14101 没有上传文件
400 14102 没有选择文件
400 14103 不支持的文件格式
400 14104 文件太大
400 14105 文件安全验证失败
500 14201 文件保存失败
500 14202 服务端缺失必要文件

2B2T.GG

GET/gg/tablist/json

返回2b2t.gg当前在线玩家列表(JSON)

该数据自2025.5.17起不再更新

请求示例

https://api.azsu.top/gg/tablist/json

GET/gg/tablist/image

返回2b2t.gg当前在线玩家列表(图片)

该数据自2025.5.17起不再更新

请求示例

https://api.azsu.top/gg/tablist/image

GET/gg/stats/<player_name>

返回指定玩家在2b2t.gg的信息

该数据自2025.5.17起不再更新

请求参数

    ‎ player_name - 玩家ID

请求示例

https://api.azsu.top/gg/stats/Az_Su

GET/gg/rankings/player/<player_name>

返回指定玩家在2b2t.gg的排名信息

该数据自2025.5.17起不再更新

请求参数

    ‎ player_name - 玩家ID

请求示例

https://api.azsu.top/gg/rankings/player/Az_Su

GET/gg/rankings/json/<type>/<player_name>

返回指定玩家在2b2t.gg中指定类型的排名信息

该数据自2025.5.17起不再更新

请求参数

    ‎ type - 类型：joins/kills/deaths/kd/playtime
    ‎ player_name - 玩家ID

请求示例

https://api.azsu.top/gg/rankings/json/playtime/Az_Su

GET/gg/rankings/json/<type>

返回2b2t.gg中指定类型的排行榜前1000名

该数据自2025.5.17起不再更新

请求参数

    ‎ type - 类型：joins/kills/deaths/kd/playtime

请求示例

https://api.azsu.top/gg/rankings/json/playtime

GET/gg/rankings/image/<type>

返回2b2t.gg中指定类型的排行榜前20名(图片)

该数据自2025.5.17起不再更新

请求参数

    ‎ type - 类型：joins/kills/deaths/kd/playtime

请求示例

https://api.azsu.top/gg/rankings/image/playtime
