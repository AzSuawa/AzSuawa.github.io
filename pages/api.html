小小素API文档 2025.9.3

公开本API提供的数据应标注来源：azsu.top

BaseURL: https://api.azsu.top

状态码

    ‎ 302 - API路径错误
    ‎ 403 - 没有访问权限
    ‎ 429 - 服务器繁忙

服务器状态查询

GET/mcping/json?ip=<ip>

返回指定MC服务器的Ping信息(JSON)

参数说明

    ‎ ip - 服务器地址

示例URL

https://api.azsu.top/mcping/json?ip=azsu.top

GET/mcping/image?ip=<ip>&watermark=<watermark>

返回指定MC服务器的Ping信息(图片)

参数说明

    ‎ ip - 服务器地址
    ‎ watermark - 自定义水印文字(可选)

示例URL

https://api.azsu.top/mcping/image?ip=azsu.top&watermark=§c延迟仅供参考§7｜§e小小素交流群:§6199755660§7｜§b制作:阿素本素§3(Az_Su)

皮肤服务

GET/skin/skin/<player_name>

返回指定ID由玩家上传的皮肤文件，若没有上传皮肤，正版ID返回正版皮肤，非正版ID返回菜鸟皮肤，用于azsu.top服务器设置皮肤功能

正版玩家皮肤来自Mojang API

参数说明

    ‎ player_name - 玩家ID

示例URL

https://api.azsu.top/skin/skin/Az_Su

2B2T.ORG数据

GET/org/rank/json/playtime/<player_name>

返回指定玩家在2b2t.org近一个月游玩时长的排名信息

基于2b2t.vc数据进行二次处理

参数说明

    ‎ player_name - 玩家ID

示例URL

https://api.azsu.top/org/rank/json/playtime/_Xiaozhen_

GET/org/rank/json/kills/<player_name>

返回指定玩家在2b2t.org近一个月击杀数的排名信息

基于2b2t.vc数据进行二次处理

参数说明

    ‎ player_name - 玩家ID

示例URL

https://api.azsu.top/org/rank/json/kills/_Xiaozhen_

GET/org/rank/json/deaths/<player_name>

返回指定玩家在2b2t.org近一个月死亡数的排名信息

基于2b2t.vc数据进行二次处理

参数说明

    ‎ player_name - 玩家ID

示例URL

https://api.azsu.top/org/rank/json/deaths/_Xiaozhen_

GET/org/rank/json/all/<player_name>

返回指定玩家在2b2t.org近一个月的全部排名信息

基于2b2t.vc数据进行二次处理

参数说明

    ‎ player_name - 玩家ID

示例URL

https://api.azsu.top/org/rank/json/all/_Xiaozhen_

2B2T.GG数据

GET/gg/tablist/json

返回2b2t.gg当前在线玩家列表(JSON)

该数据自2025.5.17起不再更新

参数说明

    ‎ <无参数

示例URL

https://api.azsu.top/gg/tablist/json

GET/gg/tablist/image

返回2b2t.gg当前在线玩家列表(图片)

该数据自2025.5.17起不再更新

参数说明

    ‎ <无参数

示例URL

https://api.azsu.top/gg/tablist/image

GET/gg/stats/<player_name>

返回指定玩家在2b2t.gg的信息

该数据自2025.5.17起不再更新

参数说明

    ‎ player_name - 玩家ID

示例URL

https://api.azsu.top/gg/stats/Az_Su

GET/gg/rank/player/<player_name>

返回指定玩家在2b2t.gg的排名信息

该数据自2025.5.17起不再更新

参数说明

    ‎ player_name - 玩家ID

示例URL

https://api.azsu.top/gg/rank/player/Az_Su

GET/gg/rank/json/<type>/<player_name>

返回指定玩家在2b2t.gg中指定类型的排名信息

该数据自2025.5.17起不再更新

参数说明

    ‎ type - 类型：joins/kills/deaths/kd/playtime
    ‎ player_name - 玩家ID

示例URL

https://api.azsu.top/gg/rank/json/playtime/Az_Su

GET/gg/rank/json/<type>

返回2b2t.gg中指定类型的排行榜前1000名

该数据自2025.5.17起不再更新

参数说明

    ‎ type - 类型：joins/kills/deaths/kd/playtime

示例URL

https://api.azsu.top/gg/rank/json/playtime

GET/gg/rank/image/<type>

返回2b2t.gg中指定类型的排行榜前20名(图片)

该数据自2025.5.17起不再更新

参数说明

    ‎ type - 类型：joins/kills/deaths/kd/playtime

示例URL

https://api.azsu.top/gg/rank/image/playtime
