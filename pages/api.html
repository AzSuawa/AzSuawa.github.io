【API维护预计过两天恢复】API仍在施工中...公开本API提供的数据应标注来源：azsu.top

BaseURL: https://api.azsu.top

状态码

    ‎ 302 - API路径错误
    ‎ 403 - 没有访问权限
    ‎ 429 - 服务器繁忙
    ‎ 5xx - 服务器内部错误

MCPing

GET/mcping/json?ip=<ip>

返回指定MC服务器的Ping信息(JSON)

请求参数

    ‎ ip - 服务器地址

请求示例

https://api.azsu.top/mcping/json?ip=azsu.top

返回参数

返回结果

http状态码 code字段 描述
200 NULL 查询成功
200 MP61 Ping查询失败
400 MP41 缺少IP参数

GET/mcping/image?ip=<ip>&watermark=<watermark>

返回指定MC服务器的Ping信息(图片)

请求参数

    ‎ ip - 服务器地址
    ‎ watermark - 自定义水印文字(可选)

请求示例

https://api.azsu.top/mcping/image?ip=azsu.top&watermark=§c延迟仅供参考§7｜§e小小素交流群:§6199755660§7｜§b制作:阿素本素§3(Az_Su)

Skin

GET/skin/skin/<player_name>

返回指定ID由玩家上传的皮肤文件，若没有上传皮肤，正版ID返回正版皮肤，非正版ID返回菜鸟皮肤，用于azsu.top服务器设置皮肤功能

正版玩家皮肤来自Mojang API

请求参数

    ‎ player_name - 玩家ID

请求示例

https://api.azsu.top/skin/skin/Az_Su

状态码

    ‎ 5xx - 服务器内部错误

2B2T.GG

GET/gg/tablist/json

返回2b2t.gg当前在线玩家列表(JSON)

该数据自2025.5.17起不再更新

请求示例

https://api.azsu.top/gg/tablist/json

GET/gg/tablist/image

返回2b2t.gg当前在线玩家列表(图片)

该数据自2025.5.17起不再更新

请求示例

https://api.azsu.top/gg/tablist/image

GET/gg/stats/<player_name>

返回指定玩家在2b2t.gg的信息

该数据自2025.5.17起不再更新

请求参数

    ‎ player_name - 玩家ID

请求示例

https://api.azsu.top/gg/stats/Az_Su

GET/gg/rank/player/<player_name>

返回指定玩家在2b2t.gg的排名信息

该数据自2025.5.17起不再更新

请求参数

    ‎ player_name - 玩家ID

请求示例

https://api.azsu.top/gg/rank/player/Az_Su

GET/gg/rank/json/<type>/<player_name>

返回指定玩家在2b2t.gg中指定类型的排名信息

该数据自2025.5.17起不再更新

请求参数

    ‎ type - 类型：joins/kills/deaths/kd/playtime
    ‎ player_name - 玩家ID

请求示例

https://api.azsu.top/gg/rank/json/playtime/Az_Su

GET/gg/rank/json/<type>

返回2b2t.gg中指定类型的排行榜前1000名

该数据自2025.5.17起不再更新

请求参数

    ‎ type - 类型：joins/kills/deaths/kd/playtime

请求示例

https://api.azsu.top/gg/rank/json/playtime
